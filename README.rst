====================================
jp_autotag
====================================

A simple program to set the title, such that our dumb mp3 player sorts the titles correctly...

Install
------------

        $ python3 -m pip install -e .


Use the tool
--------------------------

Get help:

        $ jp_autotag --help


Correct mp3 files (preview):

        $ jp_autotag *.mp3

Correct mp3 files:

        $ jp_autotag -r *.mp3
